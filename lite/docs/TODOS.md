Deferred-work ledger. Anything descoped from a sprint, discovered-but-not-done, or known
v-next work lands here **with a backlink to the originating sprint** — so nothing lives only
in a code comment or a closed conversation.

Rules:
- Every entry: `#N. [description] — deferred from S-NNN (date) — [why deferred]`.
- `/sprint create` checks this file and offers promotable items into the new sprint.
- When an item is promoted, mark it `→ promoted to S-NNN`; don't delete it.
